# 02. 复合选择器
---
1. `后代选择器`
- 后代选择器又称为包含选择器，可以选择父元素中的所有子元素。
```css
  /* 儿子元素和孙子元素都是后代 */
  parentElement childrenElement p{
    /* 样式规则 */
  }
```
2. `子选择器`
```css
  /* 子选择器只指的儿子，不指孙子 */
  parentElement childrenElement{
    /* 样式规则 */
  }
```
3. `并集选择器`
```css
  element1,element2{
    /* 样式规则 */
  }
```
